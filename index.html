jQuery Slider Demo

A script with no name, this jQuery plugin is a simple solution for implementing a slider on any web page.

Download at GitHub

Basic Setup

← Previous 1 2 3 4 5 6 7 8 Next → 
$('#slider1').slider();
$('a[href=#slider1_left]').click(function() { $('#slider1').slider('left'); return false; });
$('a[href=#slider1_right]').click(function() { $('#slider1').slider('right'); return false; });

Basic Setup w/ Infinite Loop

← Previous 1 2 3 4 5 6 7 8 Next → 
$('#slider2').slider({loop: true});
$('a[href=#slider2_left]').click(function() { $('#slider2').slider('left'); return false; });
$('a[href=#slider2_right]').click(function() { $('#slider2').slider('right'); return false; });
